# Структура проекта Legday

macOS-приложение для напоминаний вставать с места (stand-up). Живёт в меню-баре, ведёт таймер сидения/стояния, журнал и статистику.

---

## Корень проекта

| Файл / папка | Назначение |
|--------------|------------|
| **Legday/** | Исходный код и ресурсы приложения (Swift, SwiftUI, Assets). |
| **Legday.xcodeproj/** | Проект Xcode: настройки сборки, схемы, список файлов. |
| **screenshots/** | Скриншоты приложения для README и документации. |
| **scripts/** | Bash- и Python-скрипты для сборки, релиза и обработки иконок. |
| **.gitignore** | Игнорируемые Git файлы и папки (build, .env, release и т.д.). |
| **Makefile** | Цели для сборки, установки, упаковки и полного релиза. |
| **README.md** | Описание проекта и инструкции для пользователей. |
| **RELEASE.md** | Инструкции по выпуску релизов. |
| **PROJECT_STRUCTURE.md** | Этот файл: структура проекта и назначение файлов/папок. |

## Legday/ (исходный код приложения)

### Swift-файлы

| Файл | Назначение |
|------|------------|
| **StandUpApp.swift** | Точка входа приложения (`@main`). Регистрирует `StandUpAppDelegate`, создаёт `MenuBarExtra` с `MainPopupView` и лейблом в трее (`MenuBarLabel`). Обеспечивает единственный экземпляр (при повторном запуске активирует уже запущенное приложение). Запрашивает разрешение на уведомления при старте. |
| **MainPopupView.swift** | Главный UI попапа из меню-бара: заголовок, блок фазы (сижу/стою), кольцо таймера, статистика, кнопки (встал, присесть, пауза, настройки, журнал, выход). Переключает между основным экраном, настройками и журналом. Содержит стили и цвета интерфейса. |
| **StandUpState.swift** | Единый источник состояния таймера и статистики (`ObservableObject`, синглтон). Фаза сидения/стояния, оставшееся время, пауза, напоминания, счётчики (вставания за день, минуты стоя, серия дней), журнал записей. Таймер, сохранение/восстановление в UserDefaults, расчёт прогресса кольца, отложить на 15 мин, пауза, мигание иконки в трее при просроченном напоминании. |
| **SettingsStore.swift** | Хранилище настроек (синглтон, `ObservableObject`). Интервал напоминания, длительность стояния, звук, цветная иконка в трее, автозапуск при входе, рабочие часы, автостарт таймера сидения после стояния. Все значения пишутся в UserDefaults; для «запуск при входе» используется ServiceManagement. |
| **SettingsView.swift** | Не используется как отдельный экран; форма настроек встроена в `MainPopupView` как `SettingsFormContent`. Содержит UI настроек: пикеры, переключатели, кнопка «Моргнуть» для проверки мигания в трее, блок автора. |
| **ReminderWindowView.swift** | SwiftUI-вид окна напоминания «встать»: мотивационная фраза, кнопки «Встал», «+15 мин», закрыть. Вызывает колбэки в `StandUpState` (userStood, postpone15Minutes) и закрытие через `ReminderWindowManager`. |
| **ReminderWindowManager.swift** | Управление плавающим окном напоминания (NSPanel). Показ/скрытие панели с `ReminderWindowView`, центрирование, вывод поверх других окон. Синглтон. |
| **NotificationManager.swift** | Системные уведомления (UserNotifications): запрос разрешения, планирование и отмена напоминания «встать», категория с действиями «Встал» и «+15 мин». Обработка нажатий по уведомлению (UNUserNotificationCenterDelegate). |

## scripts/

| Файл | Назначение |
|------|------------|
| **do-release.sh** | Полный цикл релиза: сборка Release, упаковка .app в zip (версия = годмесяцденьчас), создание тега, создание/обновление GitHub Release и загрузка архива. Читает `GITHUB_TOKEN` из `.env` в корне. |
| **upload-release.sh** | Отдельная загрузка уже собранного zip (например `Legday-macOS-1.0.0.zip`) в существующий GitHub Release. Требует `GITHUB_TOKEN`. |
| **fix_icon.py** | Python-скрипт для обработки иконки 1024×1024: фиолетовый фон, увеличение фигуры по центру. По умолчанию берёт `Legday/Assets.xcassets/AppIcon.appiconset/icon_1024.png`. |

## Сборка и установка (корень)

- **make build** — сборка Release.
- **make run** — собрать и запустить приложение.
- **make install** — собрать и скопировать Legday.app в /Applications.
- **make zip** — упаковать текущую сборку в zip в папку `release/`.
- **make release** — полный релиз (вызов `scripts/do-release.sh`).

Артефакты сборки: папка **build/** (derivedData), при необходимости **release/** с Legday.app и zip-архивами.
